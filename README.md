## Installation
Mac
```sh
$  brew install memphis-cli
```

Linux
```sh
$ sudo apt-get install memphis-cli
```

Windows
```sh
$ choco install memphis-cli
```

### Creating a Memphis function template
```sh
$ mem func init myExampleFunc --lang nodejs
```
Supported languages: nodejs/go/python
